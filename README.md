UNIZAR LMS - Final Demo (React + Vite + Tailwind)
- Includes dark mode toggle (top-right) persisted to localStorage.
- Dummy data in frontend; results stored in localStorage.
- Assets (logo + book icons) are SVG files in src/assets/

Instructions:
1. unzip the project
2. open terminal in project folder
3. run: npm install
4. run: npm run dev
5. open the shown local URL in your browser
